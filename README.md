<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

Install docker on your machine

### Installation

Installation steps in order to use it locally and work on some features improvements

1. Clone the repo
   ```sh
   git clone https://github.com/Soluce-Technologies/portabase
   ```
3. Start the project
   ```sh
   docker compose up 
   ```
